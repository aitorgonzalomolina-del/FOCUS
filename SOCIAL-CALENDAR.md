## Estrategia WhatsApp — Reglas de Oro

| Regla | Descripción |
|-------|-------------|
| **Frecuencia** | Máx 2 mensajes por semana (martes + jueves) |
| **Horario** | 11 AM o 7 PM — nunca de noche |
| **Formato** | Siempre: emoji + título en negrita + contenido corto + CTA claro |
| **Exclusividad** | Siempre ofrecer algo exclusivo al grupo ("solo para acá", "precio especial") |
| **No abusar** | Nunca mandar 2 días seguidos ni fines de semana salvo excepción |
| **Interacción** | Pedirles que respondan (genera engagement y marca "activos") |
| **Catálogo** | Tener un PDF o galería lista para mandar cuando alguien diga "SI" |
