**Herramienta:** Gemini Imagen (Google AI Studio / Gemini app)
**Estilo de marca:** Premium · Urbano · Minimalista · Aspiracional

---

## Cómo usar estos prompts

1. Abrí **Google AI Studio** (aistudio.google.com) o la app **Gemini**
2. Seleccioná el modelo de imagen (Imagen 3 o el más reciente disponible)
3. Copiá el prompt exacto, reemplazá los [corchetes] con tu info específica
4. Generá 4 variantes y elegí la mejor
5. Descargá y publicá directamente

## TIPS PARA MEJORES RESULTADOS

| Tip | Detalle |
|-----|---------|
| **Siempre agregar** | "professional photography", "commercial quality", "editorial style" al final |
| **Para oscuro/premium** | Agregar "dark background", "moody lighting", "luxury aesthetic" |
| **Para cálido/lifestyle** | Agregar "golden hour", "warm tones", "natural light", "bokeh" |
| **Para minimalista** | Agregar "negative space", "clean composition", "minimal props" |
| **Evitar** | "person" si no querés personas, "text" si no querés texto generado |
| **Resolución** | Siempre pedir "4K" o "high resolution" |
| **Marca de referencia** | Podés agregar "similar to [Nomad/Kith/Byredo] brand photography" |

## FLUJO DE TRABAJO RECOMENDADO

```
1. Tomá una foto real del producto (aunque sea con el celu)
2. Usá Gemini para generar el fondo/ambiente ideal (prompts de arriba)
3. Si Gemini permite: subí tu foto + prompt para retocar el entorno
4. Descargá la imagen generada
5. Editá en Canva o CapCut (agregá logo, texto, ajuste de color)
6. Publicá
```
